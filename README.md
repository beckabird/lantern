# Lantern

Mobile dating app (Expo + Supabase + Stripe). See [PROJECT_PLAN.md](PROJECT_PLAN.md) for the full plan.

## Setup

```bash
npm install
```

**Note:** Run `npm install` once to generate `package-lock.json` so CI (`npm ci`) can run.

## Scripts

- `npm start` — start Expo dev server
- `npm run lint` — run ESLint
- `npm run typecheck` — run TypeScript check
